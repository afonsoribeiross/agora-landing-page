# Landing Page - Ágora Engenharia Jr.

Página web desenvolvida para a Ágora Engenharia Jr., empresa júnior de Engenharia Mecânica da UEMA (Universidade Estadual do Maranhão).

## Sobre a Ágora Engenharia Jr.
Fundada em 2017, a Ágora Engenharia Jr. é um projeto de extensão da UEMA composto por acadêmicos de Engenharia Mecânica, com foco no desenvolvimento profissional e entrega de soluções em engenharia.

## Serviços apresentados na página
- Projetos mecânicos
- Prototipagem de impressão 3D
- Projetos de climatização / PMOC
- Criação de planos de manutenção
- Projetos de exaustão
- Ensaios mecânicos
- Tratamentos térmicos

## Tecnologias
- HTML5
- CSS3
